# URL Exploder

A small single-page tool that takes any URL you paste in and explodes it into its constituent parts: scheme, userinfo, host, port, path, query parameters, and fragment. Hover over each component to explore details, host classification (DNS / IPv4 / IPv6), and links to the relevant RFCs and WHATWG sections.

Useful for quickly inspecting a complex URL, teaching the URL spec, or just satisfying curiosity about what each piece of a link actually means.

**Live preview:** [urlexploder.randomcrap.co.za](https://urlexploder.randomcrap.co.za/)
